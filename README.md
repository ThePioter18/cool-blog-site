# Cool Blog Site

I used Local WP to create a local WordPress development environment on my computer.

Inside a Local WP project, navigate to:

app/public/wp-content/themes/

and place the files from this repository there as a new WordPress theme folder.

Then open the theme folder in Visual Studio Code and run:

```bash
npm install
npm run start
```

## Tech Stack

- WordPress
- PHP
- JavaScript
- Tailwind CSS
- Local WP
- Visual Studio Code

## Features

- Responsive layout
- Custom WordPress theme
- Tailwind CSS integration
- Local WordPress development environment
- Modern frontend workflow
